## 交互流程

### 修改密码流程

1. 用户点击"安全设置"标签
2. 点击"修改密码"按钮
3. 弹出修改密码对话框
4. 输入原密码、新密码、确认新密码
5. 点击"确认修改"
6. 系统验证原密码
7. 验证新密码格式（至少8个字符）
8. 验证两次新密码是否一致
9. 调用 API 修改密码
10. 显示成功提示
11. 关闭对话框

### 查看流量统计流程

1. 用户点击"流量统计"标签
2. 系统自动加载最近7天的流量数据
3. 显示统计卡片和趋势图
4. 用户可选择不同的时间范围
5. 图表自动更新

### 撤销所有会话流程

1. 用户点击"安全设置"标签
2. 点击"撤销所有登录会话"按钮
3. 弹出确认对话框
4. 用户确认操作
5. 调用 API 撤销所有 refresh tokens
6. 显示成功提示
7. 1.5秒后自动跳转到登录页

## 响应式设计

- 桌面端：标签页横向排列，图表完整显示
- 平板端：标签页横向排列，图表适应宽度
- 移动端：标签页可滚动，图表垂直布局
