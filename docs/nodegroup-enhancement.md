# 节点组管理增强功能

## 概述

节点组管理页面已进行全面升级，新增了批量操作、快速复制、健康检查和实时监控等功能，大幅提升管理效率和可观测性。

## 新增功能

### 1. 批量操作

**功能**：
- 批量启用节点组
- 批量禁用节点组
- 批量删除节点组
- 复选框选择多个节点组
- 显示已选数量

**使用方法**：
1. 在节点组列表中勾选需要操作的节点组
2. 点击顶部的批量操作按钮
3. 确认操作

**特点**：
- 智能过滤（只操作需要的节点组）
- 操作前二次确认（删除操作）
- 并发执行，提高效率
- 操作完成后自动刷新列表

### 2. 快速复制

**功能**：
- 一键复制现有节点组配置
- 自动跳转到创建页面
- 配置预填充

**使用方法**：
1. 点击节点组操作列的"更多"按钮
2. 选择"复制"
3. 系统自动跳转到创建页面并填充配置
4. 修改名称和其他配置（可选）
5. 创建新节点组

**应用场景**：
- 创建相似配置的多个节点组
- 快速部署多个环境
- 测试不同配置参数

### 3. 健康检查

**功能**：
- 显示节点组健康状态
- 节点在线率统计
- 在线/离线节点数量
- 健康状态告警

**位置**：节点组详情页 → 健康检查标签页

**健康状态判断**：
- 健康（绿色）：在线率 ≥ 80%
- 警告（橙色）：在线率 50% - 80%
- 异常（红色）：在线率 < 50%

**告警提示**：
- 异常状态：显示红色告警，建议立即处理
- 警告状态：显示橙色提示，建议关注

**统计指标**：
- 健康状态
- 在线节点数
- 离线节点数
- 在线率百分比
- 节点状态分布

### 4. 实时监控面板

**功能**：
- 可视化展示节点组流量趋势
- 实时统计数据
- 自动刷新（每30秒）

**位置**：节点组详情页 → 实时监控标签页

**监控指标**：
- 入站流量总计
- 出站流量总计
- 总连接数
- 在线节点数
- 流量趋势图（最近24小时）
- 平均流量
- 平均连接数

**图表特性**：
- 平滑折线图
- 区域填充
- 悬停显示详细数据
- 自动刷新数据

## 注意事项

### 批量操作

- 批量删除不可恢复，请谨慎操作
- 批量启用/禁用会自动过滤不需要操作的节点组
- 建议分批操作大量节点组（避免超时）

### 快速复制

- 需要修改节点组名称（避免重复）
- 配置会完全复制
- 新节点组默认为启用状态

### 健康检查

- 数据基于节点组统计信息
- 在线率计算：在线节点数 / 总节点数
- 告警阈值可根据需求调整

### 实时监控

- 数据每30秒自动刷新
- 流量趋势图显示最近24小时
- 目前使用模拟数据（待后端 API 实现）
