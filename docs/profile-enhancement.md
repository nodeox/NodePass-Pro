# 个人中心增强功能

## 概述

个人中心页面已进行全面升级，新增了多个实用功能，提升用户体验和账户安全性。

## 新增功能

### 1. 标签页布局

个人中心现在采用标签页布局，包含三个主要模块：

- **基本信息**：查看账户基本信息和流量配额
- **安全设置**：管理密码、会话和安全选项
- **流量统计**：可视化展示流量使用趋势

### 2. 修改密码

**位置**：安全设置标签页

**功能**：
- 验证原密码
- 设置新密码（至少8个字符）
- 密码确认验证

**API 端点**：`PUT /api/v1/auth/password`

### 3. 修改邮箱（开发中）

**位置**：基本信息标签页

**功能**：
- 输入新邮箱地址
- 密码验证
- 邮箱格式验证

**状态**：前端界面已完成，后端 API 待实现

### 4. 流量使用统计图表

**位置**：流量统计标签页

**功能**：
- 时间范围选择（默认最近7天）
- 入站/出站流量趋势图
- 流量统计卡片：
  - 入站流量总计
  - 出站流量总计
  - 计费流量总计
- 使用 ECharts 实现的平滑折线图

**API 端点**：
- `GET /api/v1/traffic/usage` - 获取流量汇总
- `GET /api/v1/traffic/records` - 获取流量详细记录

### 5. 账户安全设置

**位置**：安全设置标签页

**功能**：
- Telegram 绑定状态显示
- 两步验证状态（功能开发中）
- 撤销所有登录会话
  - 一键登出所有设备
  - 需要重新登录

**API 端点**：`POST /api/v1/auth/revoke-all`

### 6. 增强的基本信息展示

**新增显示项**：
- 角色标签（管理员/普通用户）
- 账户状态标签
- VIP 等级标签
- 流量使用进度条
- 剩余流量计算


## 使用说明

### 修改密码

1. 进入个人中心
2. 切换到"安全设置"标签
3. 点击"修改密码"按钮
4. 输入原密码和新密码
5. 确认提交

### 查看流量统计

1. 进入个人中心
2. 切换到"流量统计"标签
3. 选择时间范围（可选）
4. 查看图表和统计数据

### 撤销所有会话

1. 进入个人中心
2. 切换到"安全设置"标签
3. 点击"撤销所有登录会话"按钮
4. 确认操作
5. 系统将自动跳转到登录页

   - 实现后端 API

2. **两步验证**

5. **性能优化**
   - 图表数据缓存
   - 懒加载标签页内容

## 注意事项

- 修改密码后不会自动登出其他设备，需要手动撤销会话
- 流量统计数据按小时聚合，可能存在延迟
- 撤销所有会话后需要重新登录所有设备
