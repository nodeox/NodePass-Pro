- 检查所有组件版本一致性

- 交互式版本更新

## 📊 技术实现

### 版本配置文件

**version.yaml**：
- YAML 格式，易于阅读和编辑
- 包含版本号、兼容性矩阵、更新日志
- 作为单一数据源（Single Source of Truth）

### 版本检查工具

**check-version.sh**：
- Bash 脚本实现
- 读取所有组件版本
- 比较版本一致性
- 彩色输出，易于识别

### 版本同步工具

**sync-version.sh**：
- 交互式操作
- 使用 sed 命令批量替换
- 自动更新所有组件
- 可选 Git 标签创建

### Makefile 集成

- 简化命令调用
- 统一操作接口
- 易于 CI/CD 集成
