# 隧道管理增强功能

## 概述

隧道管理页面已进行全面升级，新增了批量操作、编辑、复制和流量统计等实用功能，大幅提升管理效率。

## 新增功能

### 1. 批量操作

**功能**：
- 批量启动隧道
- 批量停止隧道
- 批量删除隧道
- 复选框选择多个隧道
- 显示已选数量

**使用方法**：
1. 在隧道列表中勾选需要操作的隧道
2. 点击顶部的批量操作按钮
3. 确认操作

**特点**：
- 支持跨页选择
- 操作前二次确认（删除操作）
- 并发执行，提高效率
- 操作完成后自动刷新列表

### 2. 编辑隧道

**功能**：
- 修改隧道名称和描述
- 更新监听端口
- 调整负载均衡策略
- 修改转发目标配置

**使用方法**：
1. 点击隧道操作列的"更多"按钮
2. 选择"编辑"
3. 修改配置
4. 保存更新

**限制**：
- 不能修改协议类型
- 不能修改入口/出口节点组（需要重新创建）

### 3. 快速复制

**功能**：
- 一键复制现有隧道配置
- 自动生成新名称（原名称 + "副本"）
- 监听端口自动留空（系统自动分配）

**使用方法**：
1. 点击隧道操作列的"更多"按钮
2. 选择"复制"
3. 修改名称和其他配置（可选）
4. 创建新隧道

**应用场景**：
- 创建相似配置的多个隧道
- 测试不同配置参数
- 快速部署多个环境

### 4. 流量统计图表

**功能**：
- 可视化展示隧道流量使用趋势
- 入站/出站流量分别统计
- 支持自定义时间范围
- 实时数据更新

**位置**：隧道详情页 → 流量统计标签页

**统计指标**：
- 入站流量总计
- 出站流量总计
- 总流量
- 按天/小时的流量趋势图

**图表特性**：
- 平滑折线图
- 区域填充
- 悬停显示详细数据
- 自适应时间轴

## 使用指南

### 批量启动隧道

1. 勾选需要启动的隧道（可跨页选择）
2. 点击顶部"批量启动"按钮
3. 系统并发启动所有选中的隧道
4. 显示成功提示

### 编辑隧道配置

1. 找到需要编辑的隧道
2. 点击"更多" → "编辑"
3. 修改配置项：
   - 名称
   - 描述
   - 监听端口
   - 负载均衡策略
   - 转发目标
4. 点击"确定"保存

### 快速复制隧道

1. 找到需要复制的隧道
2. 点击"更多" → "复制"
3. 系统自动填充配置（名称添加"副本"后缀）
4. 修改名称或其他配置（可选）
5. 点击"确定"创建

### 查看流量统计

1. 进入隧道详情页
2. 切换到"流量统计"标签页
3. 选择时间范围（默认最近7天）
4. 查看图表和统计数据

## 注意事项

### 批量操作

- 批量删除不可恢复，请谨慎操作
- 批量操作失败时会显示错误信息
- 建议分批操作大量隧道（避免超时）

### 编辑隧道

- 编辑时不能修改协议类型
- 不能修改节点组关联
- 监听端口修改可能导致连接中断

### 复制隧道

- 监听端口会自动留空（系统分配）
- 需要确保名称唯一
- 复制后的隧道默认为停止状态

### 流量统计

- 数据可能存在延迟（按小时聚合）
- 大时间范围可能影响加载速度
- 目前使用模拟数据（待后端 API 实现）
