## v0.2.0 (2026-03-07) - 重大更新

### 🎉 新增功能

#### 1. 域名绑定系统
- ✨ **自动域名绑定**: 首次验证时自动绑定网站域名
- ✨ **域名匹配验证**: 防止授权码在多个网站间共享
- ✨ **域名更换管理**: 支持管理员更换域名（30天冷却期）
- ✨ **通配符支持**: 支持 `*.example.com` 匹配所有子域名
- ✨ **测试域名豁免**: localhost、*.test 等测试域名不受限制
- ✨ **域名锁定**: 支持预设域名，防止自动绑定
- ✨ **IP 追踪**: 记录域名对应 IP，检测异常变更
- ✨ **绑定历史**: 完整的域名变更审计日志

#### 2. 现代化 Web 界面
- ✨ **React 18 + TypeScript**: 完整的前端项目
- ✨ **Ant Design 5**: 企业级 UI 组件
- ✨ **8个功能页面**:
  - 登录页面（渐变背景设计）
  - 仪表盘（实时统计 + 图表）
  - 授权码管理（完整 CRUD）
  - 套餐管理
  - 告警管理
  - Webhook 管理
  - 标签管理
  - 验证日志
- ✨ **响应式设计**: 支持移动端访问
- ✨ **暗色侧边栏**: 现代化导航体验
- ✨ **实时数据刷新**: TanStack Query 数据管理

#### 3. 监控告警系统
- ✨ **实时监控面板**:
  - 授权码统计（总数、活跃、过期、吊销）
  - 机器绑定统计
  - 验证请求统计（今日、本周、本月）
  - 即将过期提醒
- ✨ **验证趋势分析**: 7/30/90 天趋势图表
- ✨ **Top 客户统计**: 快速了解主要客户
- ✨ **自动告警**:
  - 授权码即将过期（提前30天）
  - 机器配额超限
  - 域名不匹配（可能盗用）
  - 域名 IP 变更
- ✨ **告警管理**: 标记已读、批量操作、级别筛选

#### 4. Webhook 通知
- ✨ **事件推送**: 支持 8 种事件类型
  - license.created - 授权码创建
  - license.expired - 授权码过期
  - license.revoked - 授权码吊销
  - license.transferred - 授权码转移
  - license.domain_bound - 域名绑定
  - license.domain_unbound - 域名解绑
  - alert.created - 告警创建
  - * - 所有事件
- ✨ **签名验证**: HMAC-SHA256 签名保护
- ✨ **重试机制**: 自动重试失败请求
- ✨ **日志记录**: 完整的推送日志

#### 5. 扩展功能
- ✨ **授权码转移**: 支持客户间转移，完整日志
- ✨ **批量操作**: 批量吊销、恢复、删除
- ✨ **标签管理**: 彩色标签分类授权码
- ✨ **自定义字段**: JSON 格式元数据支持
- ✨ **绑定历史**: 完整的操作审计追踪

#### 6. 安全增强
- ✨ **API 限流**: 令牌桶算法，防止暴力攻击
- ✨ **请求签名**: HMAC-SHA256 签名验证
- ✨ **IP 白名单**: 支持精确 IP 和 CIDR 网段
- ✨ **防重放攻击**: Nonce 机制
- ✨ **JWT 认证**: 安全的用户认证

#### 7. 性能优化
- ✨ **Redis 缓存**: 可选的缓存支持
- ✨ **数据库索引**: 优化查询性能
- ✨ **代码分割**: 前端按需加载
- ✨ **并发控制**: 合理的定时任务
